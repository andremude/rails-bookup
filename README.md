## Tech Stack
   * Ruby '2.7.4'
   * Rails '6.1.6'
   * PostgreSQL '12.9'
   * React "^18.2.0"

## Setup

Clone repository: `git@github.com:andremude/rails-bookup.git`

<br>

Install dependencies: 

  `bundle install`
  
  `npm install`

<br>

Installed Gems: 
  ```
  gem 'devise'
  gem 'bootstrap', '~> 5.0.0'
  gem 'faker'
  gem 'rspec-rails', '~> 6.0.0'
  gem 'dockerfile-rails', '~> 1.4'
  ```

<br>

Database management:

  `rails db:create`

  `rails db:migrate`

  `rails db:seed`

<br>  

Run server

  `rails server`

<br>

## RSpec Testing
  
 In terminal: 
  
  run `bundle exec rspec`
  
  In a new terminal tab run `./bin/webpack-dev-server`
